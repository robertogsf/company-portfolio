# NexaForge Studio Portfolio

Sitio estatico para GitHub Pages de una empresa desarrolladora de software.

## Editar contacto

El boton de WhatsApp se configura en `script.js`:

```js
const WHATSAPP_NUMBER = "5355555555";
```

Usa el numero en formato internacional, sin `+`, espacios ni guiones.

## Publicacion

El sitio no requiere build. GitHub Pages puede servirlo desde la rama `master` y la carpeta raiz.
